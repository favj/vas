What’s New in version 2.0?
Reject and adjust Invoice Line Items, Increased device support (iPad Mini and Windows 8 Phones), Searching on the Invoice and Line Item Lists.
I can’t see any of my matter information, why not?
The module for matters has not been built yet, we hope to include this functionality in a future release. To view matter information, please log into the desktop version of TyMetrix 360°.
Can I undo my adjustments?
No. If you need to perform this function you must log into the desktop version of TyMetrix 360°.
I accidentally approved an invoice. Can I undo the approval?
No, this is not allowed
I am a law firm or vendor that has been allowed to login, but all I can do is view the FAQs and support information, why is that?
This first version of the TyMetrix 360° mobile application is not intended for law firm or vendor use. We do plan to include features for you in the near future.
In my network instead of Approve and Adjust buttons we have an ‘Approve and Forward’ and ‘Adjust Net’ buttons, is this functionality included in the mobile app?
No. If you need to perform these functions you must log into the desktop version of TyMetrix 360°.
I am working on an invoice that has taxes associated with it, and I want to make an adjustment, why do I see an error message when I tap on the adjust button?
Just as in the desktop version of the app, in order to make an adjustment on an invoice with taxes you must issue a credit note. If you need to perform this function you must log into the desktop version of TyMetrix 360°.
In the desktop version, I am able to define which screen I see after I approve or reject an invoice (such as next invoice summary, next invoice line item details or stay on the same invoice), can I do this on the mobile app as well?
Not at this time. Once you approve an invoice the mobile application will bring you to the invoice summary of the next invoice pending your approval.
In the desktop version of the application, there are dynamic fields that help me review the invoice, will they be available in the mobile version of the application?
Yes
I’m used to seeing a currency selector icon on my invoice screens; does this functionality exist in the mobile version of the application?
Yes. On the homepage of the mobile application, click on Settings to switch between your preferred and original currency for reviewing invoices.
I’m used to seeing the credit notes added by the previous reviewer; does this functionality exist in the mobile version of the application?
No. If you need to see the credit notes added by the previous reviewer you must log into the desktop version of TyMetrix 360°.
